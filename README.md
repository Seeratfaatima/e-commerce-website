# Sowears Fashion - E-Commerce Website

A modern and responsive e-commerce platform built for **Sowears**, a fashion brand specializing in stylish and modest Western clothing for women. This project demonstrates a full frontend implementation with shopping cart functionality, product catalogs, and a dynamic user interface.

## üöÄ Features

*   **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
*   **Dynamic Product Catalog:** Browse products with categories, prices, and details.
*   **Shopping Cart:** Add items to the cart, update quantities, and view the total price.
*   **Product Details:** dedicated pages for each product with related item suggestions.
*   **Blog Section:** specific blog posts about fashion trends (e.g., Summer 2024 styles).
*   **Interactive UI:** Smooth transitions, sidebar navigation for mobile, and hover effects.
*   **Newsletter Subscription:** Functional input form for user engagement.
